ReminderBot
A free, lightweight desktop reminder app for Windows. No sign-up, no internet required, no ads — just simple reminders that pop up on your screen.

✨ Features

Lives quietly in your system tray (bottom right corner)
Right-click to add a reminder in seconds
Reminds you with a desktop popup on the right day
Works completely offline after install
Free forever — no accounts, no subscriptions


💻 Installation (30 seconds)

Press Win + S → search PowerShell → open it
Paste this command and press Enter:

powershellirm https://raw.githubusercontent.com/YOUR_USERNAME/reminderbot/main/install.ps1 | iex
That's it! ReminderBot will appear in your system tray immediately.

📖 How to Use
Add a reminder:

Right-click the ReminderBot tray icon → Add Reminder → type your message → type how many days from now

View upcoming reminders:

Right-click the tray icon → View Reminders

On the due date:

1 hour after you turn on your PC, a popup will appear reminding you!


🗑️ Uninstall

Press Win + R → type shell:startup → delete the ReminderBot shortcut
Press Win + R → type %appdata% → delete the ReminderBot folder


🛠️ Requirements

Windows 10 or 11
PowerShell (already installed on all Windows PCs)
